# YTAgeFree Channel DNA

Visual identity & screen-type catalog for the YouTube channel **YTAgeFree** — практические видеогайды по уходу за пожилыми людьми. Партнёр: благотворительный фонд **«Старость в радость»** (starikam.org).

## Mission

Помочь тем, кто ухаживает за пожилыми близкими, — простыми, конкретными ответами от практиков фонда. Без пафоса. С достоинством для всех сторон.

## Target audience

- Дети взрослых родителей 30–55 лет, у которых родители стали зависимыми
- Молодые соцработники и волонтёры — обучение через короткие практические видео
- Опекуны и сиделки — повышение квалификации
- Сотрудники профильных НКО

## Tone of voice

- **Эмпатичный, не сладкий.** Конкретика > эмоциональные манипуляции.
- **Уважительный к пожилым.** Никогда не «бабушки» / «дедушки» сверху вниз.
- **Личный, человеческий.** Истории и опыт > статистика и инфографика.
- **Спокойный.** Тема тяжёлая, поэтому ритм медленный, шрифт крупный, кадры дольше.

## Channel partner

**Фонд «Старость в радость»** — крупнейший в РФ НКО, работающий с домами престарелых и одинокими пожилыми. Действует с 2007 года. Сайт: https://starikam.org

Цветовая логика: **тёплый персиковый (логотип фонда) + 4 individual speaker color** — каждый спикер мгновенно узнаваем по цветовой плашке lower-third.

### Typography

- **Headings**: Montserrat 700/900 (с фоллбэком на system-ui)
- **Body**: Inter / system-ui 400/500
- **Mono / timecodes**: ui-monospace, Menlo

Исходник: `SVR_Logo_colors.ai` — векторный логотип фонда «Старость в радость» (Yandex Disk reference).

## Screen types

См. `index.html` для интерактивного каталога. Ключевые типы:

1. **`talking_head`** — спикер крупным планом (Лиля, Юра)
2. **`lower_third`** — плашка снизу с именем и ролью спикера (цветовой бортик по спикеру)
3. **`section_title`** — центрированный заголовок темы (для перехода)
4. **`step_cards`** — горизонтальные карточки шагов («Тарелка → ложка → коврик»)
5. **`split_screen`** — спикер + b-roll справа (демонстрация инструментария)
6. **`broll_only`** — только b-roll с подписью
7. **`quote_card`** — большая цитата от спикера на фоне
8. **`end_card`** — финальный кадр с CTA («подпишись», ссылка на фонд)
9. **`disclaimer_plate`** — текстовая плашка дисклеймера
